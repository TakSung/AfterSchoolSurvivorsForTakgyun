# Status: pending
# Dependencies: 8, 9
# Priority: medium
# Description: 체력바, 경험치바, 아이템 인벤토리 등 핵심 게임 UI를 구현합니다.
# Details:
src/ui/ 디렉토리에 UISystem을 구현하여 HUD 요소들을 관리합니다. HealthBar, ExperienceBar, InventoryUI, ItemSelectionUI 클래스를 만들어 각각의 UI 요소를 담당하게 합니다. pygame을 사용한 텍스트 렌더링과 프로그레스 바 표시를 구현하며, 레벨업 시 아이템 선택 모달 UI를 만듭니다. 데미지 숫자와 레벨업 이펙트 등 시각적 피드백을 추가합니다.

# Test Strategy:
UI 요소들이 게임 상태 변화에 따라 정확히 업데이트되는지 테스트합니다. 아이템 선택 UI의 상호작용과 시각적 피드백이 올바르게 작동하는지 검증합니다.

## 1. UISystem 기본 구조와 HUD 관리 시스템 구현 [pending]
### Dependencies: None
### Description: UI 요소들을 통합 관리하는 UISystem 클래스와 기본 HUD 구조를 구현합니다.
### Details:
src/ui/ui_system.py에 UISystem 클래스를 생성하여 모든 UI 요소들을 중앙에서 관리합니다. HUD 레이아웃 설정과 UI 요소들의 위치 계산을 담당하며, 게임 상태 변화에 따른 UI 업데이트 이벤트 처리 시스템을 구현합니다. pygame의 Surface와 Rect를 활용한 기본 렌더링 구조를 설정합니다.

## 2. HealthBar와 ExperienceBar 프로그레스 바 구현 [pending]
### Dependencies: 12.1
### Description: 플레이어 체력과 경험치를 표시하는 프로그레스 바 UI를 구현합니다.
### Details:
src/ui/health_bar.py와 experience_bar.py에 각각 HealthBar, ExperienceBar 클래스를 구현합니다. 현재값/최대값 비율에 따른 바 채우기 효과와 색상 변화를 구현하며, 부드러운 애니메이션 효과를 추가합니다. 체력 감소 시 빨간색 효과, 경험치 증가 시 파란색 효과 등 시각적 피드백을 포함합니다.

## 3. InventoryUI와 ItemSelectionUI 모달 시스템 구현 [pending]
### Dependencies: 12.1
### Description: 아이템 인벤토리와 레벨업 시 아이템 선택 모달 UI를 구현합니다.
### Details:
src/ui/inventory_ui.py와 item_selection_ui.py를 구현합니다. InventoryUI는 현재 보유 아이템들을 격자 형태로 표시하고, ItemSelectionUI는 레벨업 시 3개 아이템 중 선택할 수 있는 모달을 구현합니다. 마우스 클릭 이벤트 처리와 아이템 아이콘, 설명 텍스트 렌더링을 포함하며, 모달 배경 어둡게 처리와 애니메이션 효과를 추가합니다.

## 4. 시각적 피드백과 이펙트 시스템 구현 [pending]
### Dependencies: 12.2, 12.3
### Description: 데미지 숫자, 레벨업 효과 등 게임 내 시각적 피드백 시스템을 구현합니다.
### Details:
src/ui/visual_effects.py에 VisualEffects 클래스를 구현합니다. 플로팅 데미지 텍스트(위로 올라가며 페이드아웃), 레벨업 시 화면 전체 빛나는 효과, 아이템 획득 시 알림 메시지를 구현합니다. 파티클 시스템을 활용한 이펙트와 타이밍 기반 애니메이션 관리 시스템을 포함하며, 성능 최적화를 위한 이펙트 풀링 시스템을 적용합니다.
